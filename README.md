# Arms Control Research: Public Opinion and Policy Alignment

## Overview

This repository contains the research project *"Public Opinion and Arms Control Policy: A Historical Analysis (1945â€“2022)."* The study examines how U.S. public opinion has shaped and responded to arms control agreements concerning weapons of mass destruction (WMD), particularly nuclear arms. The project integrates historical polling data, media coverage analysis from *The New York Times*, and policy decisions to assess trends in sentiment and policy alignment.

## Data Sources

The datasets analyzed in this research are derived from:

- **Polling Data**: Collected from the Roper Center for Public Opinion Research, the Odum Institute, and other archives.
- **Media Coverage**: Retrieved from *The New York Times* archive API, filtering front-page articles from 1945â€“2021.
- **Policy Decisions**: Historical records of arms control treaties and agreements.
